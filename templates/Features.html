ViscoMap

This project is motivated to provide a web application to calculate the complexity of the whole website with components. This project aims to use the ViscoMap and compute and analyze the complexity of web pages on a given website.

The main objective is to develop a web application that will take the URL of a website, crawl the available pages on that site, calculate the ViscoMap score for each web page and visualize the results. This would allow the developers and website owners to see the complexity analysis of their website.
