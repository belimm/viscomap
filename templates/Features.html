Let me introduce you ViscoMap, ViscoMap is a tool that can calculate visual complexity of a webpage by crawling the main URL. If you have a problem with the crawler in your mind, you can look at the this URL: (https://en.wikipedia.org/wiki/Web_crawler). The main objective is to develop a web application that will take the URL of a website, crawl the available pages on that site, calculate the ViscoMap score for each web page and visualize the results. This would allow the developers and website owners to see the complexity analysis of their website.
